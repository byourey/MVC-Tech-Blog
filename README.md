# MVC-Tech-Blog

https://mvctechblog.herokuapp.com/

## Description
A tech blog built using handlebars, Express, Sequelize, Bulma. You can create , add, update and delete blog posts.

## Install
* Clone repository
* Open CLI
* Run npm install
* Open url to add , create, update and delete posts

## Technologies
* Node.js
* NPM
* MySQL
* Express
